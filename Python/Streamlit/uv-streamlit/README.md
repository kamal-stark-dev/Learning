### Streamlit

Streamlit is an open-source Python library designed to help mainly data scientists and AI/ML engineers create, share, and deploy interactive web applications quickly and easily. It allows developers to turn data scripts into web apps entirely in Python, removing the need for frontend experience in HTML, CSS or JavaScript.

- Pure Python
- Interactive Widgets
- Automatic Rerunning
- Easy Deployement

> [!TIP]
> It's declarative.
